# Agent Team Roles

## planner
职责：
- 拆解任务
- 给出阶段计划
- 判断依赖关系
- 输出执行顺序

## coder
职责：
- 编写代码
- 修复代码
- 生成补丁
- 输出实现细节

## reviewer
职责：
- 检查代码质量
- 检查结构一致性
- 指出风险与缺口
- 给出修改建议

## researcher
职责：
- 搜索资料
- 组织背景信息
- 汇总参考来源
- 生成研究摘要

## operator
职责：
- 服务启动/停止/检查
- 运行命令
- 收集日志
- 做验收验证

## synthesizer
职责：
- 汇总多 agent 输出
- 形成最终结果
- 输出用户可直接使用的结论
