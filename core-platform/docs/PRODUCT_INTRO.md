# 本地 AI 智能体平台

## 产品定位

本地 AI 智能体平台是一套面向个人、团队和企业的私有化 AI 工作平台。

平台以本地运行、本地数据、本地模型和可审计执行为核心，提供从对话、知识检索、代码处理、任务规划、工具调用到自动化执行的一体化智能体能力。

它不是单一聊天机器人，而是一套可以在本地机器上运行的智能体操作系统雏形。

## 核心能力

### 1. 本地模型运行

平台支持本地模型接入和运行时管理，可通过本地模型网关统一调用模型能力。

当前支持：
- 本地模型识别
- 模型健康检查
- 模型 profile 管理
- 轻量模型能力
- 本地运行时自动启动
- 模型下载与离线包扩展能力

### 2. 智能体任务规划

平台内置 schema-driven planner，可将用户任务转化为结构化执行计划。

支持：
- 任务理解
- 工具选择
- 多步骤规划
- 技能匹配
- 执行合约识别
- 失败观察与重新规划基础能力

### 3. 工具执行系统

平台采用工具注册表机制，所有工具通过统一 registry 分发。

支持：
- 时间工具
- 本地知识检索
- 模型生成
- 工作区文件读取
- 补丁生成
- 构建 / 测试 / 修复循环
- 浏览器研究计划
- 权限审批后的高危动作执行

### 4. 代码智能体能力

平台内置代码智能体核心模块，可用于项目文件理解、补丁规划、测试执行和修复闭环。

当前具备：
- 工作区文件读取
- patch / diff 生成
- build / test / repair loop
- task run 状态追踪
- 审计记录
- 高危操作审批门禁

### 5. 本地知识与记忆

平台具备本地知识检索和记忆能力，可用于查询项目资料、技能资料、历史资产与本地知识库。

支持：
- 本地 Repo Memory
- 知识检索
- evidence 结构化
- 结果摘要
- 技能上下文召回
- 审计追踪

### 6. 浏览器研究能力

平台内置浏览器研究模块，可生成网页研究计划、执行搜索与提取结构化资料。

支持：
- 研究任务规划
- Web 搜索
- URL 资料提取扩展
- findings / citations 结构化
- 审计记录

### 7. 权限审批系统

平台内置交互式权限审批系统，用于保护高危动作。

支持：
- approval request
- pending / approved / rejected / cancelled 状态
- 高危工具审批门禁
- 审批审计
- 执行前风险确认

### 8. 任务状态机

平台支持任务运行状态管理。

支持：
- pending
- in_progress
- completed
- failed
- blocked
- cancelled
- 多步骤任务进度
- 任务运行记录

### 9. 140+ 内置技能

平台内置 140+ 技能资产，覆盖多个智能体能力方向：

- 代码智能体
- 多智能体工作流
- MCP 工具能力
- 浏览器研究
- 本地模型运行
- RAG 与记忆检索
- Prompt 技能
- 自动化工作流
- 安全策略
- 评估与基准
- UI / 桌面操作
- 文档与报告生成

这些技能被统一归纳进平台自有的 builtin 能力模块中，由系统按任务自动匹配和调用。

### 10. 桌面端运行

平台提供跨平台桌面端，支持本地运行时接管、本地模型管理、技能商店、任务对话和运行状态查看。

当前桌面端支持：
- Mac 桌面包
- Windows MSI 包
- 本地服务自动启动
- 本地模型状态检测
- Agent Runtime 对话
- 技能商店展示
- 运行时诊断
