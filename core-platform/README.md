# Local AI Platform

本地 AI 智能体平台是一套可在本地机器运行的私有化 AI Agent 工作平台。

它面向个人、团队和企业，提供本地模型、本地知识、智能体规划、工具执行、代码处理、浏览器研究、权限审批和审计追踪能力。

## 核心能力

- 本地模型运行与模型网关
- Agent Runtime 智能体执行链路
- Schema-driven Planner
- Executor Tool Registry
- 代码智能体能力
- 本地知识与记忆检索
- 浏览器研究能力
- 任务状态机
- 权限审批系统
- 140+ 内置技能
- Mac / Windows 桌面端

## 当前可演示

- 本地 AI 对话
- 当前时间查询
- 项目知识查询
- 代码智能体任务
- 浏览器研究任务
- 技能商店
- 权限审批流程
- 审计报告
- Mac / Windows 桌面安装包

## 产品定位

这不是单一聊天机器人，而是一套本地 AI 智能体操作平台。

目标是让 AI 不只是回答问题，而是在本地环境中理解任务、选择工具、执行步骤、记录过程，并在关键动作前进行安全审批。
